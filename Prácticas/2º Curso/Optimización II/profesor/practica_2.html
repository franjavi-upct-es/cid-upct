2.1. Regresión lineal paso a paso (from scratch)¶

# Generamos 100 instances con una sóla característica en el intervalo (0, 2)

X = 2 * np.random.rand(100, 1)

# Generamos 100 labels 

y = 4 + 3 * X + np.random.rand(100, 1)

# dibujamos los datos generados

import matplotlib.pyplot as plt

plt.scatter(X, y)

plt.show()

Resolvemos el problema de regresión lineal

$$ \text{Minimizar en } \theta : \quad MSE (\theta) = \frac{1}{m}\sum_{i=1}^{m}\left(\theta^T x^{(i)} - y^{(i)}\right)^2. $$

donde $x^{(i)}$ es la fila i-ésima de $X$ e $y^{(i)}$ la componente i-ésima del vector de labels $Y$.

El sistema de optimalidad asociado es $$X^TX \theta = X^T Y$$ cuya solución $\theta_{\text{best}}$ viene dada por

$$ \theta_{\text{best}} = (X^TX)^{-1} X^T Y . $$
In [ ]:
# Procedemos con la optimización 
# Resolvemos la condición de optimalidad.
# Antes de nada añadimos la columna de unos para el bias

X_b = np.c_[np.ones((100, 1)), X]

# Ahora sí resolvemos la condición de optimalidad

theta_best = np.linalg.inv(X_b.T.dot(X_b)).dot(X_b.T).dot(y)

print(f"theta_best = {theta_best}")

theta_best = [[4.45624907]
 [3.01033864]]

Una vez hemos calculado el bias y el peso óptimos, definimos el modelo de predicción

In [ ]:
def model_predict(x):
    y_hat = theta_best[0] + x * theta_best[1]
    return y_hat

Finalmente, dibujamos de nuevos los datos junto con el modelo de predicción

2.2. Regresión lineal a través de algunos módulos de Python¶

Debido al intenso uso que se hace en Ciencia de Datos de la regresión lineal, ésta aparece implementada en varios módulos de Python. Veamos algunos de ellos.

2.2.1. Ajuste de mínimos cuadrados en Numpy¶

Aparece implementado en la clase scipy.linalg.lstsq donde se resuelve el sistema de optimalidad via la inversa de Moore-Penrose (también llamada pseudo-inversa). Resolvemos el problema anterior con este método.

In [ ]:
theta_best_svd, residuals, rank, s = np.linalg.lstsq(X_b.T.dot(X_b), X_b.T.dot(y), rcond=1e-6)

print(f"theta_best_svd =  {theta_best_svd}")

# rcond es el umbral para eliminar los valores singulares con valor menor que recond.
# véase la API para los detalles del resto de outputs.

theta_best_svd =  [[4.45624907]
 [3.01033864]]

Como decíamos antes, este método se basa en el cálculo de la inversa de Moore-Penrose numpy.linalg.pinv . En efecto:
