Práctica 6: métodos de gradiente y gradiente con momento paso a paso (from scratch)¶

En esta práctica estudiaremos los métodos de gradiente y gradiente con momento para el caso de la función objetivo

$$ f(x,y) = \frac{1}{2} (x^2 + b y^2) $$

con $0<b<1$. El gradiente viene dado por $$ \nabla f(x,y) = (x, by) $$ y la matriz Hessiana $$ H(x,y) = \left[\begin{array}{cc} 1 & 0 \\ 0 & b \end{array} \right] $$ tiene número de condicionamiento $cond(H) = 1/b$. La función objetivo es estrictamente convexa.

Empezamos definiendo la función objetivo y su gradiente.

import numpy as np
#b = 0.01
#b = 0.9

def objective(x):
    return 0.5 * (x[0] ** 2 + b * x[1] ** 2)

def gradiente(x):
    return np.asarray([x[0], b * x[1]])

Dibujamos la función objetivo para $b=0.01$ y $b=0.9$.

6.1 Método de gradiente¶

Implementamos el método de gradiente determinista

In [13]:
b = 0.01
def gradient_descent(objective, initial_guess, gradient, n_iter, step_size):
    solution = initial_guess
    iterations, solutions, scores = list(), list(), list()
    for i in range(n_iter):
        gradient = gradiente(solution)
        solution = solution - step_size * gradient 
        solution_eval = objective(solution)
        # ahora almacenamos las soluciones y los valores de la función objetivo
        iterations.append(i)
        solutions.append(solution)
        scores.append(solution_eval)
        print('>%d f(%s) = %.5f' % (i, solution, solution_eval))
    return [iterations, solutions, scores]

6.2 Método de gradiente con momento¶

In [15]:
def gradient_descent_moment(objective, initial_guess, initial_velocity, 
                            gradient, n_iter, step_size_sol, step_size_vel):
    solution = initial_guess
    velocity = initial_velocity
    iterations, solutions, scores = list(), list(), list()
    for i in range(n_iter):
        gradient = gradiente(solution)
        velocity = step_size_vel * velocity - step_size_sol * gradient
        solution = solution + velocity
        solution_eval = objective(solution)
        # ahora almacenamos las soluciones y los valores de la función objetivo
        iterations.append(i)
        solutions.append(solution)
        scores.append(solution_eval)
        print('>%d f(%s) = %.5f' % (i, solution, solution_eval))
    return [iterations, solutions, scores]
