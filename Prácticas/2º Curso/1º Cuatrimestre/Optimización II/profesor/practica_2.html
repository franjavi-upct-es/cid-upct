Resolvemos el problema de regresin lineal

$$ \text{Minimizar en } \theta : \quad MSE (\theta) = \frac{1}{m}\sum_{i=1}^{m}\left(\theta^T x^{(i)} - y^{(i)}\right)^2. $$

donde $x^{(i)}$ es la fila i-sima de $X$ e $y^{(i)}$ la componente i-sima del vector de labels $Y$.

El sistema de optimalidad asociado es $$X^TX \theta = X^T Y$$ cuya solucin $\theta_{\text{best}}$ viene dada por

$$ \theta_{\text{best}} = (X^TX)^{-1} X^T Y . $$
In [ ]:
# Procedemos con la optimizacin 
# Resolvemos la condicin de optimalidad.
# Antes de nada aadimos la columna de unos para el bias

X_b = np.c_[np.ones((100, 1)), X]

# Ahora s resolvemos la condicin de optimalidad

theta_best = np.linalg.inv(X_b.T.dot(X_b)).dot(X_b.T).dot(y)

print(f"theta_best = {theta_best}")

def model_predict(x):
    y_hat = theta_best[0] + x * theta_best[1]
