Consideremos la funcin $f(t) = j \frac{1}{1 + t}e^{ j (t + \frac{\pi}{2})}, \quad 0\leq t\leq 4\pi .$

Se pide:

  1. Calcula el mdulo y la fase (o argumento principal) de $f(t)$ en papel. En Python toma $t=1$.

  2. Calcula la forma binmica de $f(t)$ en papel. En Python toma $t=1$.

  3. Calcula $\frac{1}{f(t)}$ y expresa el resultado en forma binmica en papel. En Python toma $t=1$.

  4. Calcula $f^2(t)$ y $f(t) + \frac{1}{f(t)}$ y expresa el resultado en forma binmica en papel. En Python toma $t=1$.

  5. Haz un grfico $f$, es decir representa en el eje $X$ la parte real de $f$ y en el eje $Y$ la parte imaginaria. Hazlo slo en Python.

  6. Haz un grfico de $\text{Re}f(t)$ e $\text{Im}f(t)$. Hazlo slo en Python.

In[32]:
# Completar aqu
import matplotlib.pyplot as plt

def f(t):
    return 1j * (1 / (1 + t)) * np.exp(1j * (t + np.pi / 2))

t = 1

# 1)
print(f"Mdulo de f(t) en t=1: {np.abs(f(t))}")
print(f"Argumento de f(t) en t=1: {np.angle(f(t))}")

# 2)
print(f"Forma binmica de f(t) con t=1: {np.real(f(t))} + {np.imag(f(t))}j")

# 3)
f_inv = 1/f(t)

print(f"Forma binmmica de f(t) inverso con t=1: {np.real(f_inv)} + {np.imag(f_inv)}j")

# 4)
print(f"f(t)^2: {np.real(f(t)**2)} + {np.imag(f(t)**2)}j")
print(f"f(t) + 1/f(t): {np.real(f(t) + f_inv)} + {np.imag(f(t) + f_inv)}j")

# 5)
# Valores de t
t = np.linspace(0, 4*np.pi, 400)

# Se calcula el valor de f(t) para cada t
f_val = f(t)

plt.figure()
plt.plot(np.real(f_val), np.imag(f_val), color='#007aff')
plt.title('Grfico de f(t)')
plt.xlabel('Parte real')
plt.ylabel('Parte imaginaria')
plt.show()

# 6)
# Graficar Re(f(t)) y Im(f(t)) por separado
plt.figure()
plt.plot(t, np.real(f_val), label='Re(f(t))', color='#007aff')
plt.plot(t, np.imag(f_val), label='Im(f(t))', color='r', linestyle='--')

plt.title('Grfico de Re(f(t)) e Im(f(t))')
plt.xlabel('t')
plt.ylabel('Valor')
plt.legend()
plt.show()
