Receta de Relleno para pastel de tres leches.
Para hacer la clásica torta tres leches simplemente debes tener listo un bizcocho y la crema correspondiente para bañarlo. Pero, ¿cómo se hace esa crema? En RecetasGratis te mostramos los ingredientes y el paso a paso que debes seguir para elaborar un relleno para pastel de tres leches delicioso y perfecto para cualquier tipo de queque. Verás que los productos empleados son de fácil acceso y que el proceso de elaboración es tan sencillo que hasta una persona que no esté acostumbra a preparar este tipo de postres puede conseguir un excelente resultado. ¡Anímate y cuéntanos tu experiencia!
En una cacerola a fuego bajo se pone a calentar la leche condensada, la leche evaporada y la media crema. Se mezcla bien para que se integren correctamente. Aparte, separa las claras de las yemas de los huevos.
Cuando la mezcla de las tres leches esté tibia, se añaden las yemas y se sigue mezclando, el azúcar y la vainilla. Sigue batiendo para que, poco a poco, se vaya formando la crema.
Después, se pasa un poco de la mezcla a la licuadora para batirla con la maicena. Hecho esto, se vuelve a pasar a la cacerola junto con el resto de la crema de leches.
Mueve la mezcla sin parar para evitar que se pegue y, cuando haya espesado lo suficiente, se retira del fuego. Aparte, se baten las claras hasta alcanzar el punto de nieve y se integran a la mezcla tibia haciendo movimientos envolventes.
¡Ya tienes listo tu relleno para pastel de tres leches! Ahora, puedes emplearlo para bañar cualquier bizcocho de vainilla, de limón, o del sabor que más te guste.
