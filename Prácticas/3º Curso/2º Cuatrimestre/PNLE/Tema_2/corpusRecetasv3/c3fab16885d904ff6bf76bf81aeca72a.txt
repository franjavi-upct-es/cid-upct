Receta de Ceviche mexicano de pescado.
Esta receta reproduce como nuestros abuelos hacían el Ceviche y la podemos considerar la auténtica receta original del Ceviche de México. Verás que es una receta muy rica y cuando comas este ceviche los sabores estallarán en tu boca. También puedes probar en Recetasgratis.net otros tipos de Ceviches, por ejemplo el ceviche ecuatoriano.
Se cortan a cuadritos pequeños el pescado y se ponen en un recipiente de cristal junto con el jugo de los limones exprimidos y sin pepitas. Es muy importante que limpiemos el limón de pepitas, también podemos usar limas. Esta es la base de cualquier Ceviche, a partir de ahora le daremos a nuestro plato el toque Mexicano.
Se deja cocer el pescado en el limón media hora, pasado la media hora se cuela el pescado y se le agrega toda la verdura, cilantro, apio, tomate, cebolla y pimiento picado.
Se le agrega solo unas hojitas de orégano, sal y el jugo magui.
Se revuelve sin estropear el ceviche, aquí es muy importante hacerlo con delicadeza para que el pescado no se vea machacado.
Se deja en la nevera hasta el momento se servir, también se le puede agregar gambas o camarones pequeños pelados, pulpo cocido, o almejas, es la variedad que usted quiera.
Se sirve el ceviche original mexicano acompañado de tostas o de galletas saladas o craquers y salsa picante. Este Ceviche es un buen entrante...... buen provecho.
