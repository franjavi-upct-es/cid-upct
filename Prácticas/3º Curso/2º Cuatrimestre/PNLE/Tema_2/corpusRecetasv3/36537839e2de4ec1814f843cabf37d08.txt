Receta de Salsa roja mexicana.
La salsa roja mexicana es todo un clásico de esta gastronomía, ya que se utiliza para condimentar una infinidad de platos. Así, podemos utilizar la salsa roja para nachos, para tacos, para enchiladas, y un largo etcétera. Ahora bien, si lo que no sabes hacer es, precisamente, esta salsa, ¡a continuación lo descubrirás!En RecetasGratis damos cabida a las cocinas de todo el mundo y compartimos las recetas propias de cada país, las más innovadoras e, incluso, las de nueva creación. Por ello, la tradicional salsa roja mexicana no podía faltar. Si quieres saber cómo hacerla, ¡sigue leyendo!
Primero ponemos todo en el comal para asarlo, moviendo constantemente para que no se quemen los ingredientes y la salsa roja mexicana casera pueda adquirir el sabor perfecto.
Cuando esté todo bien asado, molemos los alimentos con el perejil. Podemos llevar a cabo este paso en una licuadora o utilizar una batidora eléctrica de mano.
Servimos la salsa roja mexicana, sazonamos al gusto y machacamos el aguacate.
¡Servimos! :3 Esta salsa roja mexicana es perfecta para condimentar tacos, sopes y prácticamente cualquier platillo.
Si no sabes qué tacos preparar para utilizar tu salsa roja mexicana casera, toma nota de las siguientes sugerencias:Tacos dorados de papa con polloTacos de arrachera.
